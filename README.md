# MJ-SYSTEM Highrise Bot

24/7 Highrise bot with music, emotes, wallet, and moderation.

## Features
- 🎵 Music queue
- 🎭 Emote loops
- 💰 Wallet & tips
- 🛡️ Moderation

## Commands
- !play [song] - Add song
- !queue - Show queue
- !skip - Skip song
- !tip [amount] - Send tip
- 1 - Start emotes
- 0 - Stop emotes
- !help - Show all commands
